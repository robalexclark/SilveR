Boxplot

On the boxplot the median is denoted by the horizontal line within the box. The box indicates the interquartile range, where the lower and upper quartiles are calculated using the type 2 method, see Hyndman and Fan (1996). The whiskers extend to the most extreme data point which is no more than 1.5 times the length of the box away from the box. Individual observations that lie outside the outlier range are included on the plot using stars.

Analysis conclusions

The difference between groups C and A is statistically significant at the 1% level of significance as the p-value is less than 0.01 (Steel's test).

Analysis description

The comparison of treatment factor levels back to a single control group was made using the non-parametric Steel's test, see Steel (1959).

Non-parametric tests should be used if the data is non-normally distributed, the variability of the responses varies across treatments or the responses are not continuous and numerical.

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

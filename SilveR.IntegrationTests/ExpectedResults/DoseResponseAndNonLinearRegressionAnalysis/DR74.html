Description of the analysis results

The data was analysed using non-linear regression. The estimate of the minimum of the curve is 0.009. The estimate of the maximum of the curve is 1.004. The estimate of the logED50 of the curve is -3.240. The back-transformed estimate of the ED50 of the curve is 0.039. The slope of the curve was fixed at 1.

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

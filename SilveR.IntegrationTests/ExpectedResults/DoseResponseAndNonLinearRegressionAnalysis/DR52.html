Description of the analysis results

The data was analysed using non-linear regression. The estimate of the maximum of the curve is 0.986. The estimate of the slope coefficient of the curve is 1.095. The estimate of the logED50 of the curve is -3.709. The back-transformed estimate of the ED50 of the curve is 0.024. The minimum of the curve was fixed at 0.

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

Description of the analysis results

The data was analysed using non-linear regression. The estimate of the minimum of the curve is -3.651. The estimate of the maximum of the curve is -0.035. The estimate of the slope coefficient of the curve is 1.192. The estimate of the logED50 of the curve is -1.787. The back-transformed estimate of the ED50 of the curve is 0.167.

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

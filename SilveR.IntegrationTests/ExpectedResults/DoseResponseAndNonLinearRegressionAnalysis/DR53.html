Description of the analysis results

The data was analysed using non-linear regression. The estimate of the minimum of the curve is 0.017. The estimate of the slope coefficient of the curve is 1.082. The estimate of the logED50 of the curve is -1.579. The back-transformed estimate of the ED50 of the curve is 0.026. The maximum of the curve was fixed at 1.

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

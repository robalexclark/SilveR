InVivoStat Dose-response Analysis

Response and dose variables

The Resp 1 response is currently being analysed by the Dose-response and Non-Linear Regression Analysis module.

The dose variable (Dose1) has been log10 transformed prior to analysis.

Description of the analysis results

The data was analysed using non-linear regression. The estimate of the minimum of the curve is 0.031. The estimate of the maximum of the curve is 0.997. The estimate of the slope coefficient of the curve is 1.109. The ED50 of the curve was fixed at 0.03.

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

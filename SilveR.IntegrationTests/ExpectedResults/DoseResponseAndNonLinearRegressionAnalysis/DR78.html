Description of the analysis results

The data was analysed using non-linear regression. The estimate of the minimum of the curve is 0.020. The estimate of the maximum of the curve is 0.989. The estimate of the slope coefficient of the curve is -1.127. The estimate of the logED50 of the curve is -3.220. The back-transformed estimate of the ED50 of the curve is 0.040.

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

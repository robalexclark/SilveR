InVivoStat AUC Transformation

Input variables

The responses are stored in the variable Resp2, the subjects are stored in the variable Subject1 and the time points are stored in variable Time1.

The area under curves (AUC) have been calculated assuming that the time course starts at 0.

The AUC responses are calculated using the trapezoid approach.

Note that for the additional variables included in the AUC dataset, the values quoted (for each subject) correspond to the level at the initial timepoint.

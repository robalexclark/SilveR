InVivoStat AUC Transformation

Input variables

The responses are stored in the variable Resp1, the subjects are stored in the variable Subject1 and the time points are stored in variable Time1.

The area under curves (AUC) have been calculated using the change from baseline responses.

Case profiles plot including AUC

Note that in the AUC calculation, any area corresponding to a negative change from baseline will be subtracted from the AUC total.

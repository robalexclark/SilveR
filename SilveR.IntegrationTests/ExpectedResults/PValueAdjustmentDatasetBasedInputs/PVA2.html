InVivoStat P-value Adjustment

The input p-values for this module should be unadjusted p-values. These unadjusted p-values are adjusted using Hochberg's multiple comparison procedure, Hochberg (1988).

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

Selected variables

The variable Resp 1 has been selected as the response and the variable Time 1 has been selected to define the X-axis. The variable Resp 1 has been log10 transformed. The variables Cat2 and Treat 1 have been selected as the categorisation factors. The variable Animal2 defines the case profiles.

Selected variables

The variable Resp 1 has been selected as the response and the variable Time2 has been selected to define the X-axis. The variable Resp 1 has been arcsine transformed. The variables Cat1 and Cat2 have been selected as the categorisation factors. The variable Anima 1 defines the case profiles.

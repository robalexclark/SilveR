Selected variables

The variable Resp2 has been selected as the response and the variable Anima 1 has been selected to define the X-axis. The variables Cat1 and Time 1 have been selected as the categorisation factors.

Categorised boxplot

On the boxplot the median is denoted by the horizontal line within the box. The box indicates the interquartile range, where the lower and upper quartiles are calculated using the type 2 method, see Hyndman and Fan (1996). The whiskers extend to the most extreme data point which is no more than 1.5 times the length of the box away from the box.

The plot includes individual observations, denoted by black dots.

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

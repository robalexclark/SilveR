Analysis description

The data were analysed using a 1-way ANOVA approach, with PVTestGroup1 as the treatment factor. This was followed by comparisons of the predicted means of the PVTestGroup1 factor back to the control group mean using Dunnett's procedure, Dunnett (1955). This was followed by all pairwise comparisons between the predicted means of the PVTestGroup1 factor using Tukey's procedure, Braun (1994).

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

Analysis description

The data were analysed using a 3-way ANOVA approach, with Treat1, Treat2 and Treat3 as the treatment factors and Blo ck1 as a blocking factor. This was followed by all pairwise comparisons between the predicted means of the Treat1 * Treat2 * Treat3 interaction using Tukey's procedure, Braun (1994).

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

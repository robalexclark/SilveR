Analysis description

The data were analysed using a 2-way ANOVA approach, with Treat24 and Treat25 as the treatment factors. This was followed by comparisons of the predicted means of the Treat24 factor back to the control group mean using Hochberg's procedure, Hochberg (1988). This was followed by all pairwise comparisons between the predicted means of the Treat24 factor using Hochberg's procedure, Hochberg (1988).

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

Analysis description

The data were analysed using a 2-way ANOVA approach, with Treat 20 and Treat 21 as the treatment factors. This was followed by all pairwise comparisons between the predicted means of the Treat 20 * Treat 21 interaction using Hommel's procedure, Hommel (1988). The response was log10 transformed prior to analysis to stabilise the variance.

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

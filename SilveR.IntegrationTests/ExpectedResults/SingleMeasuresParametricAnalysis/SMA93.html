Analysis description

The data were analysed using a 2-way ANOVA approach, with Treat 20 and Treat 21 as the treatment factors. This was followed by all pairwise comparisons between the predicted means of the Treat 20 * Treat 21 interaction using Tukey's procedure, Braun (1994).

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

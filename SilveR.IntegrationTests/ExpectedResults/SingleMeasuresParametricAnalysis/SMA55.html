Analysis description

The data were analysed using a 1-way ANOVA approach, with Treat8 as the treatment factor. This was followed by comparisons of the predicted means of the Treat8 factor back to the control group mean using Hommel's procedure, Hommel (1988). This was followed by all pairwise comparisons between the predicted means of the Treat8 factor using Hommel's procedure, Hommel (1988).

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

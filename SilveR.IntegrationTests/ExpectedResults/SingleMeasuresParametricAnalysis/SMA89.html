Analysis description

The data were analysed using a 1-way ANOVA approach, with Treat24 as the treatment factor. This was followed by comparisons of the predicted means of the Treat24 factor back to the control group mean using Dunnett's procedure, Dunnett (1955). This was followed by all pairwise comparisons between the predicted means of the Treat24 factor using Tukey's procedure, Braun (1994). The response was log10 transformed prior to analysis to stabilise the variance.

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

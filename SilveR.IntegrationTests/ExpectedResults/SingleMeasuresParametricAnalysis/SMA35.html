Analysis description

The data were analysed using a 2-way ANOVA approach, with Treat1 and Treat2 as the treatment factors and Blo ck1 as a blocking factor. This was followed by comparisons of the predicted means of the Treat1 factor back to the control group mean using Bonferroni's procedure, Bonferroni (1936). This was followed by all pairwise comparisons between the predicted means of the Treat1 factor using Bonferroni's procedure, Bonferroni (1936).

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

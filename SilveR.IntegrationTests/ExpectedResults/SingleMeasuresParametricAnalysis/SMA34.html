Analysis description

The data were analysed using a 2-way ANOVA approach, with Treat1 and Treat2 as the treatment factors. This was followed by comparisons of the predicted means of the Treat1 factor back to the control group mean using Hommel's procedure, Hommel (1988). This was followed by all pairwise comparisons between the predicted means of the Treat1 factor using Hommel's procedure, Hommel (1988). The response was log10 transformed prior to analysis to stabilise the variance.

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

Analysis description

The data were analysed using a 1-way ANOVA approach, with Treat9 as the treatment factor. This was followed by comparisons of the predicted means of the Treat9 factor back to the control group mean using Holm's procedure, Holm (1979). This was followed by all pairwise comparisons between the predicted means of the Treat9 factor using Holm's procedure, Holm (1979).

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

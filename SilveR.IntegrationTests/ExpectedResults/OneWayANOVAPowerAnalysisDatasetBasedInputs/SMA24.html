InVivoStat 'One-way ANOVA' Power Analysis

The power calculations made by InVivoStat assume that the future experimental design involves one treatment factor, with equal group sizes. The data will be analysed using a balanced One-way ANOVA.

The statistical power generated is for the overall ANOVA test (i.e. an overall difference between the group means) .

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

Response and treatment factor

The response analysed is Resp1, with Treat2 fitted as the single treatment factor.The response has been log10 transformed prior to analysis.

Selected results

Assuming the significance level is set at 1%, the number of groups is 4, with a sample size of 2 per group, if the effect size is 0.096, then the statistical power is 7.15%.

Assuming the significance level is set at 1%, the number of groups is 4, with a sample size of 11 per group, if the effect size is 0.096, then the statistical power is 99.59%.

Assuming the significance level is set at 1%, the number of groups is 4, with a sample size of 20 per group, if the effect size is 0.096, then the statistical power is >99.99%.

Definitions

Power: The chance of detecting a statistically significant test result from running an experiment, assuming there is a real biological effect to find.

Significance level: The chance that the experiment will give a false-positive result.

Response and covariance structure

The Resp 1 response is currently being analysed by the Repeated Measures Parametric Analysis module.

The repeated measures mixed model analysis is using the compound symmetric covariance structure to model the within-subject correlations. When using this structure you are assuming sphericity and also that the variability of responses is the same at each level of Day2, see Pinherio and Bates (2002). These assumptions may not hold in practice.

Analysis description

The data were analysed using a 2-way repeated measures mixed model approach, with Treat4 as the treatment factor, Day2 as the repeated factor and Block1 as the blocking factor. This was followed by Planned Comparisons on the predicted means to compare the levels of the effect Treat4 * Day2.

The compound symmetric covariance structure was used to model the within-subject correlations. When using this structure we assumed that the variability of the responses was the same at each level of Day2 and the correlation between responses from any pair of levels of Day2 is the same.

A full description of mixed model theory, including information on the R nlme package used by InVivoStat, can be found in Venables and Ripley (2003) and Pinherio and Bates (2002).

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

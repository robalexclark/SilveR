Response and covariance structure

The Resp 1 response is currently being analysed by the Repeated Measures Parametric Analysis module.

The repeated measures mixed model analysis is using the unstructured covariance structure to model the within-subject correlations. When using this structure you are estimating many parameters. If the numbers of subject used is small then these estimates may be unreliable, see Pinherio and Bates (2002).

Analysis description

The data were analysed using a 2-way repeated measures mixed model approach, with Treat4 as the treatment factor and Day2 as the repeated factor. This was followed by Planned Comparisons on the predicted means to compare the levels of the effect Treat4 * Day2.

The unstructured covariance structure allowed the variability of the responses to be different, depending on the level of Day2. This structure also allowed the correlation between responses from any pair of levels of Day2 to be different. While this approach is the most general it should be used with care when there are few subjects, as many parameters are required to be estimated. These estimates may not be very reliable.

A full description of mixed model theory, including information on the R nlme package used by InVivoStat, can be found in Venables and Ripley (2003) and Pinherio and Bates (2002).

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

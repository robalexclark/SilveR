Analysis description

The data were analysed using a 5-way repeated measures mixed model approach, with Treat17, Treat18, Treat19 and Treat20 as treatment factors and Day3 as the repeated factor. This was followed by Planned Comparisons on the predicted means to compare the levels of the effect Treat17 * Treat18 * Treat19 * Treat20 * Day3. The response was arcsine transformed prior to analysis to stabilise the variance.

The compound symmetric covariance structure was used to model the within-subject correlations. When using this structure we assumed that the variability of the responses was the same at each level of Day3 and the correlation between responses from any pair of levels of Day3 is the same.

A full description of mixed model theory, including information on the R nlme package used by InVivoStat, can be found in Venables and Ripley (2003) and Pinherio and Bates (2002).

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

InVivoStat Repeated Measures Parametric Analysis

Response and covariance structure

The CVResp response is currently being analysed by the Repeated Measures Parametric Analysis module.

The repeated measures mixed model analysis is using the compound symmetric covariance structure to model the within-subject correlations. When using this structure you are assuming sphericity and also that the variability of responses is the same at each level of CVTime1, see Pinherio and Bates (2002). These assumptions may not hold in practice.

Only use the pairwise comparisons you planned to make a-priori, these are the so called Planned Comparisons, see Snedecor and Cochran (1989). No options are available in this module to make multiple comparison adjustments. If you wish to apply a multiple comparison adjustment to these results then use the P-value Adjustment module.

Analysis description

The data were analysed using a 2-way repeated measures mixed model approach, with CVTreat1 as the treatment factor and CVTime1 as the repeated factor. This was followed by Planned Comparisons on the predicted means to compare the levels of the effect CVTreat1 * CVTime1.

The compound symmetric covariance structure was used to model the within-subject correlations. When using this structure we assumed that the variability of the responses was the same at each level of CVTime1 and the correlation between responses from any pair of levels of CVTime1 is the same.

A full description of mixed model theory, including information on the R nlme package used by InVivoStat, can be found in Venables and Ripley (2003) and Pinherio and Bates (2002).

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

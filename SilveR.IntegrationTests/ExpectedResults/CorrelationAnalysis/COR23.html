Analysis description

This test is a rank-based measure of association and can be used when the data are not necessarily from a bivariate normal distribution, see Hollander and Wolfe (1973). The test statistic is the estimate scaled to zero mean and unit variance, and is approximately normally distributed.

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

InVivoStat Equivalence TOST Test Analysis

Response and equivalence bounds

The Resp 1 response is currently being analysed by the Equivalence (TOST) test Analysis module.The response has been loge transformed prior to analysis.

The lower equivalence bound is defined as a 10.00 % (decrease) change. The upper equivalence bound is defined as a 10.00 % (increase) change. As both boundaries are defined a two one-sided (TOST) equivalence test has been performed.

Analysis description

The data were analysed using a Two One-Sided (TOST) equivalence test, see Limentani et al. 2005, with Treat1 as the treatment factor. The response was loge transformed prior to analysis to stabilise the variance.

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

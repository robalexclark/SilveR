InVivoStat Equivalence Test Analysis

Response and equivalence bound

The Resp10 response is currently being analysed by the Equivalence (TOST) test Analysis module.

The lower equivalence bound is defined as a 10 % (decrease) change from the overall response mean. This is equivalent to an absolute change of size -0.05. As only a lower bound has been defined a one-sided equivalence test has been performed.

Analysis description

The data were analysed using a one-sided equivalence test, see Limentani et al. 2005, with Treat9 as the treatment factor.

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

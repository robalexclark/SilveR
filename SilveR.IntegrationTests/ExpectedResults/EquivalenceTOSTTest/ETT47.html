Analysis description

The data were analysed using a one-sided equivalence test, see Limentani et al. 2005, with Treat1 as the treatment factor. The response was arcsine transformed prior to analysis to stabilise the variance.

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

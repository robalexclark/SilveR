InVivoStat Within-Subject Analysis

Response and covariance structure

The CVResp response is currently being analysed by the Extended Paired t-test Analysis and Within-Subject Analysis module.

The analysis, which is an implementation of a repeated measures mixed model, is using the compound symmetric covariance structure to model the within-subject correlations. When using this structure you are assuming sphericity and also that the variability of responses is the same at each level of CVTime1, see Pinherio and Bates (2002). These assumptions may not hold in practice.

Analysis description

The data were analysed using a repeated measures mixed model approach utilizing the mmrm R package, with treatment factor CVTime1. This was followed by planned comparisons on the predicted means to compare the levels of the CVTime1.

The compound symmetric covariance structure was used to model the within-subject correlations. When using this structure we assumed that the variability of the responses was the same at each level of CVTime1 and the correlation between responses from any pair of levels of CVTime1 is the same.

The analysis implements the Kenward-Roger approximation of the degrees of freedom, see Kenward and Roger (1997).

A full description of mixed model theory can be found in Venables and Ripley (2003) and Pinherio and Bates (2002).

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

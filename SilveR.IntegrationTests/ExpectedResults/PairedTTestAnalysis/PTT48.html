Analysis description

The data were analysed using an repeated measures mixed model approach, with treatment factor Day4 and Block1 as the blocking factor. This was followed by planned comparisons on the predicted means to compare the levels of the Day4. The response was loge transformed prior to analysis to stabilise the variance.

The first order autoregressive covariance structure was used to model the within-subject correlations. When using this structure we assumed that the variability of the responses was the same at each level of Day4. We also assumed that the correlation between responses from any pair of levels of Day4 was related to the distance between them.

A full description of mixed model theory, including information on the R nlme package used by InVivoStat (beta version), can be found in Venables and Ripley (2003) and Pinherio and Bates (2002).

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

InVivoStat Within-Subject Analysis

Response and covariance structure

The Resp 1 response is currently being analysed by the Extended Paired t-test Analysis and Within-Subject Analysis module. The response has been square root transformed prior to analysis.

The analysis, which is an implementation of a repeated measures mixed model, is using the unstructured covariance structure to model the within-subject correlations. When using this structure you are estimating many parameters. If the numbers of subjects used is small then these estimates may be unreliable, see Pinherio and Bates (2002).

Analysis description

The data were analysed using a repeated measures mixed model approach utilizing the mmrm R package, with treatment factor Day1 and Block1 as the blocking factor. This was followed by planned comparisons on the predicted means to compare the levels of the Day1. The response was square root transformed prior to analysis to stabilise the variance.

The unstructured covariance structure allowed the variability of the responses to be different, depending on the level of Day1. This structure also allowed the correlation between responses from any pair of levels of Day1 to be different. While this approach is the most general it should be used with care when there are few subjects, as many parameters need to be estimated. These estimates may not be very reliable.

The analysis implements the Kenward-Roger approximation of the degrees of freedom, see Kenward and Roger (1997).

A full description of mixed model theory can be found in Venables and Ripley (2003) and Pinherio and Bates (2002).

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).
